#### Executar um arquivo JavaScript com node
    - Entrar na pasta de um arquivo .js | cd nome-da-pasta 
    - Executar um arquivo .js           | node nome-do-arquivo.js
    - Limpar tela                       | clear ou cls
    - Listar arquivos no terminal       | ls ou dir
